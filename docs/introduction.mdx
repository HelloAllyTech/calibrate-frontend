## Voice Agent Pipeline

A voice agent processes conversations through this pipeline:

```
User Speech → [STT] → Text → [LLM] → Response Text → [TTS] → Agent Speech
                              ↓
                        [Tool Calls]
                              ↓
                       External APIs
```

Pense helps you test and benchmark each component:

| Component | What Pense Does |
|-----------|-----------------|
| **STT** | Compare providers for transcription accuracy |
| **LLM** | Test responses, tool calls, benchmark models |
| **TTS** | Compare providers for voice quality |
| **Full Pipeline** | Run end-to-end simulations |

## Recommended Workflow

<Steps>
  <Step title="Evaluate STT/TTS Providers">
    Run [STT](/guides/stt) and [TTS](/guides/tts) evaluations to find the best providers for your language.
  </Step>
  <Step title="Build and Test Your Agent">
    Follow the [LLM Testing](/guides/llm-testing) guide to create an agent, add tools, write tests, and benchmark models.
  </Step>
  <Step title="Run Simulations">
    Set up [simulations](/guides/simulations) with personas and scenarios, then run text or voice tests.
  </Step>
</Steps>
